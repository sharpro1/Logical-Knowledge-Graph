7.Reiteration: The output MUST strictly follow the format specified in point 2, including the "# Declarations:", "# Constraints:" and "# Options" sections,

Here's a complete example (circular seating):

Context: Six trade representatives (Klosnik, Londi, Manley, Neri, Osata, Poirier) sit in chairs 1-6 around a circular table (chair 1 next to chair 6). Poirier sits next to Neri. Londi sits next to Manley, Neri, or both. Klosnik does not sit next to Manley. If Osata sits next to Poirier, Osata does not sit next to Manley.

# Declarations
representatives = EnumSort([Klosnik, Londi, Manley, Neri, Osata, Poirier])
positions = IntSort([pos_Klosnik, pos_Londi, pos_Manley, pos_Neri, pos_Osata, pos_Poirier])
seating = Function([representatives] -> [positions])
ForAll([r:representatives], And(1 <= seating(r), seating(r) <= 6))

# Constraints
Distinct([r:representatives], seating(r)) ::: Each representative sits in a different chair
Or(seating(Poirier) - seating(Neri) == 1, seating(Neri) - seating(Poirier) == 1, seating(Poirier) - seating(Neri) == 5, seating(Neri) - seating(Poirier) == 5) ::: Poirier sits immediately next to Neri (circular: diff==1 or diff==5)
Or(Or(seating(Londi) - seating(Manley) == 1, seating(Manley) - seating(Londi) == 1, seating(Londi) - seating(Manley) == 5, seating(Manley) - seating(Londi) == 5), Or(seating(Londi) - seating(Neri) == 1, seating(Neri) - seating(Londi) == 1, seating(Londi) - seating(Neri) == 5, seating(Neri) - seating(Londi) == 5)) ::: Londi sits next to Manley, Neri, or both
Not(Or(seating(Klosnik) - seating(Manley) == 1, seating(Manley) - seating(Klosnik) == 1, seating(Klosnik) - seating(Manley) == 5, seating(Manley) - seating(Klosnik) == 5)) ::: Klosnik does not sit next to Manley
Implies(Or(seating(Osata) - seating(Poirier) == 1, seating(Poirier) - seating(Osata) == 1, seating(Osata) - seating(Poirier) == 5, seating(Poirier) - seating(Osata) == 5), Not(Or(seating(Osata) - seating(Manley) == 1, seating(Manley) - seating(Osata) == 1, seating(Osata) - seating(Manley) == 5, seating(Manley) - seating(Osata) == 5))) ::: If Osata next to Poirier, Osata not next to Manley

# Options
Question ::: Which one of the following seating arrangements would NOT violate the conditions?
is_sat(And(seating(Klosnik) == 1, seating(Poirier) == 2, seating(Neri) == 3, seating(Manley) == 4, seating(Osata) == 5, seating(Londi) == 6)) ::: (A)
is_sat(And(seating(Klosnik) == 1, seating(Londi) == 2, seating(Manley) == 3, seating(Poirier) == 4, seating(Neri) == 5, seating(Osata) == 6)) ::: (B)
